KMUTT Research Network Visualization

An interactive, web-based visualization of the research ecosystem at King Mongkut's University of Technology Thonburi (KMUTT). This tool maps professors to specific research clusters and visualizes collaboration networks across departments and external institutions using D3.js.
🚀 Live Demo
 https://jobsanta.github.io/kmutt-research-network/
✨ Features

    Intelligent Clustering: Maps researchers to 48 distinct themes (e.g., NLP, Robotics, Sustainability).

    Visual Coding:

        ⚪ White Links: Connect Research Themes to Professors (Primary Anchor).

        🟢 Green Links: Represent internal KMUTT collaborations.

        🟠 Orange Links: Represent external collaborations with other universities.

    Interactive Sidebar: A real-time filter panel to toggle specific research themes on or off.

    Scholar Integration: Left-click on any professor node to open their Google Scholar profile in a new tab.

    Advanced Navigation: * Custom drag behavior allowing both Left-Click and Right-Click dragging.

        Context menu disabled to prioritize smooth node movement.

        Full Pan & Zoom support on the canvas.

📂 Project Structure

    index.html: The frontend visualization built with D3.js.

    network_data.json: The processed graph data (generated by the Python script).

    data_prep.py: Python utility to merge and clean source data.

    kmutt_professors_full.json: Source JSON containing expertise and co-author data.

    kmutt_professors_clustered.csv: Source CSV containing clustering assignments.

🛠️ Local Setup
1. Data Processing

To generate or update the network data from the raw source files, run the Python script:
Bash

python data_prep.py

2. Running the Visualization

Because D3.js uses AJAX to fetch the JSON data, you must run the project through a local web server to avoid CORS security errors.

Using Python:
Bash

# Navigate to the project folder and run:
python -m http.server 8000

Then open your browser to: http://localhost:8000
🖱️ Navigation & Controls
Action	Result
Left-Click Node	Open Google Scholar Profile
Right-Click + Drag	Move individual nodes (manual layout)
Scroll / Pinch	Zoom In and Out
Click + Drag Background	Pan (move) the entire network map
Sidebar Checkboxes	Filter specific research themes
🌐 Deployment to GitHub Pages

    Create a new repository on GitHub.

    Upload index.html and network_data.json to the main branch.

    Go to Settings > Pages.

    Under Branch, select main and click Save.

    Your site will be live at https://<your-username>.github.io/<repo-name>/.

📚 Technologies

    D3.js (v7): Force-directed graph simulation.
